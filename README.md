A decentralized **Web3 charity donation platform** built with **Next.js**, **TypeScript**, and **Solidity**. The platform enables transparent and secure blockchain-based donations using **MetaMask** on the **Sepolia Testnet**, with future plans for AI-powered recommendations and decentralized storage.

---

## ✨ Features

- 🔐 MetaMask Wallet Connection
- 💝 Charity Selection
- ⛓️ Solidity Smart Contract Integration
- 📊 Donation Tracking
- 🌐 Web3 Powered Donations
- ⚡ Built with Next.js 15 & TypeScript

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/SunnyAgrwl05/gasless-charity-hub.git
```

Navigate to the project directory:

```bash
cd gasless-charity-hub
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```
