# Gasless Charity Hub

A decentralized charity donation platform built using:

- Next.js
- TypeScript
- Solidity
- MetaMask
- Sepolia Testnet

## Features

- Wallet Connection
- Charity Selection
- Smart Contract Integration
- Donation Tracking
- Web3 Powered Donations
